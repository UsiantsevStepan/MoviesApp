# MoviesApp
### MoviesApp provides featured lists of movies and gives user an opportunity to search any movie. It also provides movies details including trailers, teasers and etc.
### Used MVC pattern, Core Data, non-storyboard based UI, [iOS YouTube helper](https://developers.google.com/youtube/v3/guides/ios_youtube_helper) and [The Movie Database API](https://www.themoviedb.org/documentation/api).
